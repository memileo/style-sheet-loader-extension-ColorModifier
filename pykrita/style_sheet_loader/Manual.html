Style Sheet Loader

Loads a Qt Style Sheet and applies it to the UI.

Usage

Tools->Scripts->Load Style Sheet.
Set the path to load a text file containing a Qt Style Sheet and apply it to the Krita window.
If "Load on startup" is checked, this filepath will be saved in Krita's config and loaded automatically on startup.
If "Use folder resource prefix" is checked, the given text will be used as a resource prefix for the style sheet to find resources such as images in the same folder. For example with the "stylesheet" prefix in the line "image: url(stylesheet:icon.png);".
